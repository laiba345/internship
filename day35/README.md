- location.reload()是JavaScript中的方法，**用于重新加载当前页面。它会导致整个页面被重新加载，包括所有的资源文件（如CSS、JavaScript等），并且会重置页面的状态。**
    - 这种就比较消耗页面的性能和资源了

location.reload()
