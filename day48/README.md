1. 工具栏那一块的想法

2. 对data中某一块数据的属性进行修改
- this.$set方法，参数值; 
    - 1、对象
    - 2、对象中的属性(字符串形式)
    - 3、需要赋的值
```
this.$set(item, 'isCollect', true)
```

3. forEach的操作，其实就是对数据进行相关操作
    - 用于加快遍历简易程度

4. 详解JS中的forEach方法
```
array.forEach(function(currentValue, index, array) {
    // 执行操作的代码
});
```
注意：回调函数接收三个参数: 1、当前迭代的元素值 2、当前迭代的索引 3、正在被迭代的数组本身
